### tmpfs

User-friendly bash script to mount a RAM disk in `/mnt/tmpfs` on linux systems

### Usage

```bash
# create ram disk
./tmpfs
Mounted file system at /mnt/tmpfs with size 2048M

# resize
./tmpfs resize 1024
Resized file system to 1024M

# unmount
./tmpfs unmount
Unmounted file system at /mnt/tmpfs
```
